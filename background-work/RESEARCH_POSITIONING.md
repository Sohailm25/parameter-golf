# Research Positioning

This repo is a local research and submission-engineering workspace for Parameter Golf.

## What It Is

- a disciplined local experiment system
- a place to evaluate the challenge landscape as of March 19, 2026
- a home for atomic iteration tracking, full file snapshots, and a golden set
- a bridge between local MacBook experimentation and later official submission formatting

## What It Is Not

- a claim that we already know the winning architecture
- a fork that silently drifts away from the official challenge rules
- a generic LLM research sandbox with no promotion discipline

## Success Condition

The repo succeeds if it makes the next best move obvious:

- which lane to run next
- which iteration is actually best
- why that iteration is best
- whether that result belongs to the spirit-first lane, the open-rules lane, or both
